# Windows Local Keymap
状态：执行完成

## 目的

记录这台 Windows 机器上《天涯明月刀手游》当前真实生效的键位，以及固定剧本里和潜行相关的唯一控制规则。

## 当前确认键位

- `W`：前进
- `A`：左移
- `S`：后退
- `D`：右移
- `Shift`：翻滚
- `Space`：跳跃
- `F`：交互
- `Esc`：菜单
- `M`：地图
- `N`：背包

## 当前确认技能键

- `2`：潜行阶段补 buff
- `3`：闷棍
- `4`：妙取

## 当前固定规则

- Stage 4 `dark_close`
  - 必须先点人
  - 再进潜行
  - 第一次成功进入潜行后，直接走 `3` 闷棍与搜刮链
- Stage 5 `dark_miaoqu`
  - 不点人
  - 直接进潜行后按 `4`
- 路人 NPC 社交仍然走点击链，不把 `F` 当成主链 owner
- 功能型 NPC 仍可在明确出现交互提示时使用 `F`

## 保持不变

- 这份文档记录的是本机当前真实行为，不做跨机器兼容
- 如果后续游戏内键位被改动，应直接更新本文档与对应代码，不保留旧行为分支
