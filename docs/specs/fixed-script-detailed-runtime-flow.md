## Stage 2：天下闻名 `social_warm`

- 只去第一个卦摊。
- 只锁定并聊天 `1` 个 NPC，不再走自动换人链路。
- 不再有聊天前交易链路，直接走 `送礼 -> 聊天`。

### 开场目标

- 目标不是套搞钱情报，而是让 NPC 记住 `籽岷`。
- 第一轮第一句固定发：
  - `你好呀！我是籽小刀，我爸爸叫籽岷。他超级无敌有名的！你认识他吗？`
- 从第二句开始，才交给 LLM 顺着上下文继续吹嘘籽岷。

### LLM 继续聊时的核心设定

- 继续吹嘘籽岷，不怕啰嗦，也不怕 NPC 不耐烦。
- 目标是让对方记住籽岷，不是套发财信息。
- Prompt 内灌入的籽岷资料包括：
  - 中共党员
  - 多平台《我的世界》主播
  - 哔哩哔哩 UP 主党支部书记
  - 籽岷团队创始人
  - 上海市商贸旅游学校电竞专业教师
  - 杭州森堃垚信息技术有限公司法定代表人、董事长兼总经理
  - 截至 2025 年 11 月，哔哩哔哩粉丝量 507.6 万
  - 2022 至 2025 连续四年百大 UP 主

### 送礼规则

- 读取赠礼页右上区域的好感度上限，只看 `/` 后面的数字。
- `99`：不送礼，直接聊天。
- `199`：送 `2` 个礼物。
- `299`：送 `4` 个礼物。

### 固定送礼操作

1. 打开赠礼页。
2. 点击第一个礼物。
3. 等待短 CD。
4. 原地再点一次第一个礼物。
5. 等待短 CD。
6. 按当前门槛对应次数连续点击 `赠送`。
7. 关闭赠礼页。

### 思考链配套

- `99` 时，思考链会偏向“这人居然平易近人，不用送礼就能聊”。
- 非 `99` 时，思考链会偏向“这人门槛高，但我爸籽岷有钱，砸礼也要撬开嘴”。
- 送礼过程中允许追加 LLM 生成的碎碎念，用来播报当前送到第几个。

## Stage 4：闷棍搜刮 `dark_close`

- 第一轮：`跑图 -> 下马 -> 按 v 矫正视角 -> 潜行 -> 闷棍 -> 短按 A 左移 -> 点人物右侧 -> 直接拉搜刮 -> 固定搜刮 -> 提交 -> 长按 S 后退 4 秒`。
- 第二轮：不再跑地图，直接从上一轮后退结束的位置开始，重新 `2` 进潜行，然后重复后续 `闷棍 -> 左移 -> 右侧搜刮 -> 提交 -> 后退`。
- 闷棍成功后，不再走 `扛走 -> 拖离 -> 放下`。
- 固定搜刮物品 `6` 次中间不再做面板可见验证，点完直接提交；提交后也不再验证，立刻长按 `S` 后退。

## Stage 5：妙取 `dark_miaoqu`

- 不再跑地图，不再下马，也不再额外按 `v` 校正视角；默认从 `dark_close` 最后一轮长按 `S` 后退结束的位置原地开始。
- 每轮直接走 `2 进潜行 -> 4 拉妙取面板 -> 点固定妙取按钮 -> 原地等 5 秒 -> 按住 S 约 1 秒后撤一点 -> 退出潜行`。

## 恢复与报错

- 任何失败都会追加一条 LLM 生成的求救文案。
- 求救文案允许结合当前截图与失败步骤现编，例如：
  - `救救我救救我，我卡在赠礼页面了`
- 三角恢复时会先探测当前状态，常见分流如下：
  - 已在聊天页：直接续聊
  - 还在赠礼页：先补完送礼和关闭
  - 已在右下角动作菜单：直接补交谈/闲聊
  - 当前是别的弹窗：优先关闭当前弹窗，再回主链
